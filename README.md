# E-Commerce Sales Analysis

## Project Overview

This project analyzes e-commerce sales data to identify revenue drivers, profitability trends, customer behavior, and category performance.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset

Superstore Dataset

Total Records: 9,994+

## Analysis Performed

### Data Cleaning
- Missing Value Check
- Duplicate Check
- Data Type Validation

### Feature Engineering
- Year Extraction
- Month Extraction
- Shipping Days Calculation

### Exploratory Data Analysis

- Sales Trend Analysis
- Category Analysis
- Profit Analysis
- Customer Analysis
- Regional Analysis
- Shipping Analysis
- Correlation Analysis

## Key Business Insights

- Technology category generated highest sales.
- Consumer segment contributed maximum revenue.
- High discounts negatively impacted profitability.
- Standard Class was the most preferred shipping mode.
- Top customers generated significant revenue.

## Recommendations

- Optimize discount strategy.
- Focus on high-value customers.
- Improve inventory planning.
- Monitor low-profit categories.
